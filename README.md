# Project Name 
App User Behavior Segmentation

# Objective
The App User Behavior Segmentation project applies K-Means Clustering to group users based on their app usage patterns. 

# Key Features
- User behavior segmentation using K-Means Clustering.
- Exploratory Data Analysis (EDA) and data preprocessing.
- Feature selection based on the Silhouette Score.
- Cluster visualization using PCA.

# Technologies and Tools Used
- Pandas – Data manipulation and preprocessing.
- NumPy – Numerical computations.
- Matplotlib & Seaborn – Data visualization.
- Scikit-learn – K-Means Clustering, StandardScaler, PCA, and evaluation metrics.

# Machine Learning Model
- Model Used: K-Means Clustering
- Problem Type: Unsupervised Learning (Clustering)
- Number of Clusters: 4
- Evaluation Metrics: Elbow Method and Silhouette Score
- Visualization: Principal Component Analysis (PCA)

# Project Workflow
- Data Collection
- Data Understanding
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Selection
- Data Scaling
- Optimal Cluster Selection (Elbow Method)
- K-Means Model Training
- Cluster Evaluation (Silhouette Score)
- PCA Visualization
- Cluster Profiling
